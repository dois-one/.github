# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within DOIS, please send an email to security@dois.one. All security vulnerabilities will be promptly addressed.

Please do not publicly disclose the issue until it has been addressed by the team.

## Security Practices

- All admin routes are protected by Auth0 authentication
- Environment variables are stored securely in Vercel
- No secrets are committed to the repository
- Dependencies are regularly updated for security patches
